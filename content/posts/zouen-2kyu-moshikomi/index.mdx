2級造園施工管理技士は **造園キャリアの第一歩** として最適な資格です。19歳以上であれば実務経験ゼロで受験でき、造園会社・公園管理会社・植栽工事会社の若手技術者にとって登竜門となります。

この記事では、2級造園に独学・1発合格した筆者が、**2級造園に特化した受験申込み手順** を解説します。

> 4資格まとめての比較は [施工管理技士 受験申込み完全ガイド](/posts/shikaku-moshikomi-flow) を参照。

---

## 2級造園 受験申込みの流れ

| ステップ | 時期（目安） | やること |
|---------|-----------|---------|
| 1. 受験要項確認 | 6月頃 | 全国建設研修センターHPで公表 |
| 2. 申込み | 7月上旬〜8月上旬 | インターネット推奨 |
| 3. 実務経験証明書準備 | 申込みと同時 | 会社印・所属長の証明 |
| 4. 受験票受領 | 10月中旬 | 試験会場の確定 |
| 5. **第一次検定** | **11月上旬** | 結果は1月発表 |
| 6. **第二次検定** | **11月上旬**（同日） | 結果は3月発表 |
| 7. 合格証明書 | 3月以降 | 国土交通大臣名の証明書 |

> ⚠️ 試験日程は年度で変動します。最新情報は [一般財団法人 全国建設研修センター](https://www.jctc.jp/) で必ず確認してください。
> 2級造園は **第一次・第二次が同日** に実施されます。

## 2級造園 受験資格

### 第一次検定：実務経験不要

**19歳以上**（試験実施年度内に19歳に達する者）であれば、誰でも受験可能。学生・実務経験ゼロの社会人もOK。

### 第二次検定：学歴別の実務経験要件

| 学歴区分 | 指定学科卒 | 指定学科以外卒 |
|---------|----------|------------|
| 大学 | 1年以上 | 1.5年以上 |
| 短大・高専 | 2年以上 | 3年以上 |
| 高校 | **3年以上** | 4.5年以上 |
| その他（中卒等） | 8年以上 | 8年以上 |

> ※ 令和6年度から制度改定あり。**第一次検定合格後3年以上の実務経験** ルートも追加されています。最新の要件は必ず公式要項を確認してください。

### 2級造園の「指定学科」

1級造園と同じく以下が指定学科です。

- **造園学科 / 造園科**
- **農学科 / 園芸学科 / 園芸農学科**
- **林学科 / 林業学科**
- **農業土木学科 / 土木工学科**
- **緑地計画学科**

> ⚠️ 「建築学科」は造園では指定学科に **該当しません**（土木では該当）。

## 認められる実務経験（2級造園編）

実務経験として認められる工事範囲は1級造園と同じです。

### ✅ 認められる造園工事

- **植栽工事**（街路樹・公園緑化・庭園植栽・屋上緑化）
- **移植工事**（既存樹木の保存移植）
- **造園土工 / 客土・整地**
- **石材工 / 竹垣工 / 生垣工**
- **修景施設工**（園路・広場・水景）
- **遊戯施設工**（公園遊具設置）
- **管理施設工**（造園関連境界フェンス等）

### ❌ 認められない業務

- **庭木の剪定・除草のみ**（管理業務）
- **農業従事 / 花卉栽培・販売**
- **造園設計のみ**
- **建築外構の土木部分**

> 💡 2級造園でも、**「庭木の剪定・除草」だけの経験では認められません**。植栽工事・移植工事など「造園**工事**」の施工管理経験が必要です。

## 2級造園の実務経験証明書 記入例

### ✅ 良い記入例（高卒3年想定）

| 期間 | 工事名・発注者 | 職務内容 |
|------|--------------|---------|
| 令和3年4月〜令和4年3月 | ○○公園植栽工事（○○市） | 工事担当者として、街路樹植栽工の **施工管理** を補佐 |
| 令和4年4月〜令和5年3月 | ○○団地外構緑化工事（○○県） | **現場代理人補佐** として、植栽・支柱工事の品質・安全管理を担当 |
| 令和5年4月〜令和6年3月 | ○○庭園修景工事（○○市） | **主任技術者補助** として、石組み・園路・植栽工の施工管理を実施 |

### 2級造園の職務内容で使うキーワード

- **施工管理**（工程・品質・安全）
- **現場代理人補佐** / **主任技術者補助**
- **工事担当者** / **施工担当者**
- **植栽工** / **造園土工** / **修景施設工**
- **品質管理** / **出来形管理**

> 💡 2級は経験年数が短いので、**「補佐」「補助」「担当」** レベルのキーワードで十分です。1級ほど高度な役職表現は不要。

## 2級造園 受験申込みの注意点

### 注意1：第一次・第二次が同日

2級造園は **第一次・第二次が同日** に実施されます。第二次の受験資格を満たす場合は、**同時受験で1日完結** がおすすめ。

### 注意2：年1回のみ（土木のような前期・後期はない）

2級造園は **年1回**（11月）のみの実施です。土木のように前期6月・後期10月の **年2回** ではないので、申込み時期を逃すと **1年待ち** になります。

### 注意3：庭木の手入れだけの経験ではダメ

植木屋さんで **「庭木の剪定・除草」しかしていない** 場合、実務経験として認められません。**工事の施工管理** に関わった経験のみ申告できます。

### 注意4：会場が少ないので早めの宿泊予約

造園は受験者数が少なく **会場も少ない** ため、地方在住者は前泊が必要なケースが多いです。試験日が決まったら早めに宿泊予約を。

## 2級造園の受験料・受験会場

### 受験料

- **第一次検定のみ**：7,200円程度
- **第一次・第二次同時**：14,400円程度
- **第二次検定のみ**：7,200円程度

> ※ 金額は年度で変動。最新は公式要項を確認。

### 試験会場（土木より少ない）

- 札幌・仙台・東京・名古屋・大阪・広島・高松・福岡・那覇 など主要都市のみ

## 申込み完了後にやること

1. **受験票が届いたら** 顔写真・氏名・会場を確認
2. **学習スケジュール** を確定（[2級造園 独学勉強法](/posts/zouen-2kyu-benkyoho)）
3. **参考書を購入**：2級造園は **1冊で第一次・第二次に対応** したテキストでOK（[造園施工管理技士 おすすめ参考書](/posts/zouen-sankosho)）
4. **過去問を解き始める**（無料の[過去問チャレンジ](/quiz)）

## まとめ

| 項目 | 2級造園のポイント |
|------|------------------|
| 第一次検定 | **19歳以上で実務経験不要** |
| 第二次検定の実務経験 | 高卒・指定学科で **3年以上** |
| 試験回数 | **年1回（11月）** |
| 試験日 | 第一次・第二次が **同日（11月上旬）** |
| 受験料 | 第一次のみ約7,200円／同時約14,400円 |
| 認められる工事 | 植栽・移植・修景・園路・石材工 |
| 認められない業務 | 庭木剪定・除草・農業・花卉販売 |
| 会場数 | 土木より少ない（地方は遠征覚悟） |

2級造園は **造園会社の若手・キャリア入口** に最適な資格です。年1回しかないので、毎年の申込み時期（7〜8月）を逃さないように受験要項をウォッチしておきましょう。
