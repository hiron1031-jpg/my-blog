1級土木施工管理技士の **受験申込み** は、勉強より先に立ちはだかる最初の壁です。とくに **実務経験証明書** で「職務内容を何と書けばいいのか」「会社印の依頼をいつ出すか」で詰まる人が多く、書き方を間違えると **不受理 → 1年待ち** になります。

この記事では、1級土木に独学・1発合格した筆者が、**1級土木に特化した受験申込み手順** と **実務経験証明書の書き方** を解説します。

> 4資格まとめての比較は [施工管理技士 受験申込み完全ガイド](/posts/shikaku-moshikomi-flow) を参照。

## 1級土木 受験申込みの流れ

| ステップ | 時期（目安） | やること |
|---------|-----------|---------|
| 1. 受験要項確認 | 1〜2月 | 全国建設研修センターHPで公表 |
| 2. 申込み開始 | 3月中旬〜4月上旬 | インターネット推奨 |
| 3. 実務経験証明書準備 | 申込みと同時 | **会社印・所属長の証明** |
| 4. 受験票受領 | 6月 | 試験会場の確定 |
| 5. 第一次検定 | **7月上旬** | 結果は8月発表 |
| 6. 第二次検定 | **10月上旬** | 結果は1月発表 |
| 7. 合格証明書 | 2月以降 | 国土交通大臣名の証明書 |

> ⚠️ 試験日程は年度で変動します。最新情報は [一般財団法人 全国建設研修センター](https://www.jctc.jp/) で必ず確認してください。

## 1級土木 第二次検定の受験資格

**1級土木の第一次検定は19歳以上であれば受験可能** ですが、**第二次検定** は学歴別の実務経験要件があります。

### 学歴別の必要実務経験年数

| 学歴区分 | 指定学科卒 | 指定学科以外卒 |
|---------|----------|------------|
| 大学 | **3年以上** | 4.5年以上 |
| 短大・高専 | 5年以上 | 7.5年以上 |
| 高校 | 10年以上 | 11.5年以上 |
| その他（中卒等） | **15年以上** | 15年以上 |

> ※ 令和6年度から制度改定があり、第二次検定の受験資格が緩和されました。**第一次検定合格後5年以上の実務経験** ルートも追加されています。最新の要件は必ず公式要項を確認してください。

### 1級土木の「指定学科」とは？

以下の学科を卒業していると、上記表の「指定学科卒」枠で受験できます。

- **土木工学科**
- 農業土木学科
- 建築学科
- 都市工学科
- 衛生工学科
- 交通土木工学科
- 鉱山土木工学科
- 森林土木工学科
- 砂防工学科

「自分の学科がどっち？」と迷ったら、卒業証明書を取り寄せて学科名を正確に確認してください。建築学科は土木の指定学科に **含まれます**（造園とは扱いが異なるので注意）。

## 実務経験として認められる職務（土木編）

### ✅ 認められる職務（土木工事）

- 河川工事 / 砂防工事 / 海岸工事 / 港湾工事
- 道路工事 / 舗装工事 / 橋梁工事 / トンネル工事
- 上下水道工事 / 鉄道工事 / ダム工事
- 土地造成工事 / 地盤改良工事 / 法面保護工事

これらの工事で **「施工管理」業務** を行っていれば、実務経験として認められます。

### ❌ 認められない業務

- 純粋な **設計のみ**（施工管理を伴わないもの）
- **建築工事のみ**（土木と扱いが異なる）
- **営業・経理・人事** の事務作業
- **作業員としての労務** のみ（管理業務がない）
- 学生時代のアルバイト

## 【最重要】1級土木の実務経験証明書 記入例

### ✅ 良い記入例（1級土木）

| 期間 | 工事名・発注者 | 職務内容 |
|------|--------------|---------|
| 令和2年4月〜令和3年3月 | 国道○号線改良工事（国土交通省○○地方整備局） | **現場代理人補佐** として、土工・舗装工の **施工管理**（工程・品質・安全）を担当 |
| 令和3年4月〜令和4年3月 | ○○川河川改修工事（○○県） | **主任技術者** として、護岸工・築堤工の施工管理を統括 |
| 令和4年4月〜令和5年9月 | 県道○号線橋梁補修工事（○○県） | **現場代理人** として、橋梁補修工の施工計画・品質管理を実施 |

### ❌ NG記入例

| 期間 | 工事名 | 職務内容 |
|------|------|---------|
| 令和2〜令和5年 | 道路工事複数 | 現場で作業 |

→ 期間が曖昧、工事名が抽象的、職務内容が「作業員」と読まれて不受理リスク大。

### 1級土木の職務内容で必ず使うキーワード

以下のキーワードを **職務内容欄に1つ以上** 入れてください。これがないと「施工管理業務」と判定されにくいです。

- **施工管理**（工程・品質・安全・原価）
- **現場代理人** / **現場代理人補佐**
- **主任技術者** / **監理技術者補佐**
- **工事監督員** / **工事監督補助**
- **施工計画の立案**
- **出来形管理 / 品質管理**

## 1級土木 受験申込みでよくある失敗

### 失敗1：会社印が締切に間に合わない

**対策**：申込み開始日（3月中旬）と同時に会社へ依頼。最低でも締切1ヶ月前には書類を回し始める。

### 失敗2：転職組が前職の証明を取り損ねる

**対策**：転職前の会社にも依頼が必要。退職後でも当時の会社が証明可能。**気まずさを乗り越えて頼む** こと。多くの会社は対応に慣れています。

### 失敗3：建築工事を土木の経験として申告

**対策**：建築工事は1級建築施工管理技士の経験。1級土木では認められません。土木工事との明確な区別を。

### 失敗4：指定学科の確認漏れ

**対策**：卒業証明書を取り寄せて学科名を確認。「建設学科」「環境工学科」など曖昧な学科は事前に問い合わせ。

## 1級土木の受験料・受験会場

### 受験料

- **第一次検定**：10,500円程度
- **第二次検定**：10,500円程度
- **第一次・第二次同時受験**：21,000円程度

> ※ 金額は年度で変動。最新は公式要項を確認。

### 試験会場

1級土木は受験者数が多く、**全国主要都市** に会場があります。
- 札幌・仙台・東京・新潟・名古屋・大阪・広島・高松・福岡・那覇 など

希望会場を申込み時に選択できます（先着順で埋まる場合あり）。

## 申込み完了後にやること

1. **受験票が届いたら** 顔写真・氏名・会場を確認
2. **学習スケジュール** を確定（[1級土木 独学勉強法](/posts/doboku-1kyu-benkyoho)）
3. **参考書を購入**（[土木施工管理技士 おすすめ参考書](/posts/doboku-sankosho)）
4. **過去問を解き始める**（無料の[過去問チャレンジ](/quiz)）

## まとめ

| 項目 | 1級土木のポイント |
|------|------------------|
| 第一次検定 | 19歳以上で受験可（実務経験不要） |
| 第二次検定の実務経験 | 大卒・指定学科で **3年以上** |
| 指定学科 | 土木工学・農業土木・建築 等 |
| 試験日 | 第一次：7月／第二次：10月 |
| 認められる工事 | 道路・河川・橋梁・トンネル・上下水道 等 |
| 必須キーワード | 施工管理・現場代理人・主任技術者 |
| 申込み締切 | 3月中旬〜4月上旬（会社印は1ヶ月前から準備） |

申込みでつまずかなければ、あとは勉強するだけ。受験要項が発表されたら、**まず実務経験証明書の準備から動き出す** のが合格への第一歩です。
