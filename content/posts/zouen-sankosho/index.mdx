「造園のテキスト、どこ行っても売ってない…」

2級造園の勉強を始めようと近所の書店に行ったときの話です。資格コーナーに行っても、土木や電気・建築のテキストは何冊も並んでいるのに、造園施工管理技士の棚はほぼ空。1冊あるかないか。

ネットで調べても「おすすめテキスト」の情報が少なく、口コミもほとんどない。試験勉強以前に、「どの教材を買えばいいのか」からつまずいてしまう——これが造園受験者の最初の壁です。

**私も同じ経験をしました。** 造園作業員として働きながら2級造園・1級造園を受験した際、教材探しに本当に苦労しました。この記事では、その苦労から得た「造園施工管理技士の正しい教材の選び方と使い方」を全てお伝えします。

---

## 造園施工管理技士の教材事情【なぜ少ないのか】

まず現実を知っておいてください。造園施工管理技士のテキストは、土木や建築と比べて**圧倒的に少ない**です。

**理由は受験者数にあります。**

1級土木施工管理技士の年間受験者数が数万人規模であるのに対し、1級造園施工管理技士は数千人程度にとどまります。出版社にとっては「作っても売れない」分野なので、新しいテキストが生まれにくいのです。

この現実は、ある意味で合格のチャンスでもあります。

**「教材が少ない＝みんなが同じ教材を使っている」**

つまり、正しい教材さえ選べば、あとはやり込んだ量がそのまま合否に直結する試験です。教材選びで迷う時間を減らして、勉強時間を最大化することが造園合格の鍵です。

## おすすめテキスト比較

### 第1位：[例題で学ぶ!! 2級造園施工管理技術検定 第1次検定（弘文社）](https://www.amazon.co.jp/dp/4770328524)（新品購入）

> **おすすめ度：★★★★★**

私が実際に**新品で購入し、メインテキストとして使った**一冊です。造園施工管理技士専門のテキストで、内容が試験に直結しているのが大きな強みです。

| 項目 | 内容 |
|------|------|
| 形式 | 例題（過去問）＋解説 |
| 入手方法 | 書店・Amazon（新品推奨） |
| こんな人に向いている | 試験に出る内容に絞って効率よく学びたい人 |

**良かった点：**
- 実際の試験と同じ形式の例題で学べるため、実戦感覚がつかみやすい
- 造園施工管理技士に特化した内容で、不要な情報が少ない
- 植物・法規・施工管理の各分野がバランスよくカバーされている

**注意点：**
- 植物のカラー図版は少ないため、植物の外見は別途覚える工夫が必要
- 全部やろうとせず、頻出分野に絞って使うのが正解

**使い方のコツ：** まず例題を解いてみる → 間違えた問題の解説を丁寧に読む → 関連する知識をテキストで補強、というサイクルで回す。最初から読もうとしない。

### 第2位：日建学院テキスト（中古で入手）

> **おすすめ度：★★★★☆**

私が**中古で安く入手してサブテキストとして使った**一冊。日建学院は建設系資格の専門スクールで、そのノウハウが詰まったテキストです。

| 項目 | 内容 |
|------|------|
| 形式 | 解説テキスト＋問題集 |
| 入手方法 | メルカリ・ヤフオク等の中古（1,000〜2,000円程度） |
| こんな人に向いている | 別の角度から理解を深めたい人・1冊では不安な人 |

**良かった点：**
- メインテキストで理解できなかった箇所が別の解説で「腑に落ちる」ことがある
- 問題の解き方・解説のアプローチが異なるため、視野が広がる
- 中古なら安く手に入るのでコスパがいい

**注意点：**
- 古い版だと法改正前の内容が含まれる場合がある（法規の数値は最新情報で確認）
- あくまでサブとして使うこと。メインは1冊に絞る

**使い方のコツ：** 「例題で学ぶ」をメインに置いて、理解できない部分だけこちらで補う。2冊を並行してやろうとすると時間が足りなくなるので注意。

## 過去問集の活用法【センター公式を必ず使う】

造園施工管理技士の過去問には、大きく2種類のルートがあります。

**① 市販のテキスト付属の過去問**

[「例題で学ぶ!! 2級造園施工管理技術検定」](https://www.amazon.co.jp/dp/4770328524)などのテキストには過去問が収録されています。解説が充実しており、「なぜその答えか」まで理解しながら学習を進められます。まずこちらから始めるのがおすすめです。

**② 全国建設研修センター公式の問題集**

試験実施機関が発行しているため、本試験に最も近い出題形式です。センターのウェブサイトから直接購入可能。試験本番の感覚をつかむには最適です。

**私が実践した使い方：**

| 時期 | 使用教材 | 目的 |
|------|---------|------|
| 試験3ヶ月前〜 | [例題で学ぶ!! 2級造園施工管理技術検定（弘文社）](https://www.amazon.co.jp/dp/4770328524)（メインテキスト） | 全体像の把握・基礎知識の習得 |
| 試験2ヶ月前〜 | テキスト付属の過去問を繰り返す | 弱点分野の集中演習 |
| 試験1ヶ月前〜 | センター公式問題集 | 本試験形式に慣れる |
| 試験直前 | ×マーク問題のみ | 弱点の最終確認 |

**「5年分か10年分か」という問題ですが、造園の場合は5〜7年分を3周することをおすすめします。**

土木と同様、造園も過去問の繰り返し出題率が高い試験です。同じ問題が表現を変えて出題されることも多い。5〜7年分を3周した方が、10年分を1周するより確実に力がつきます。

## 植物・法規知識の補強方法【テキスト以外の勉強法】

造園施工管理技士の合格で最大の壁となるのが**植物知識**です。テキストだけでは不足する部分を、どう補うか。

### 植物図鑑・樹木カードの活用

テキストに掲載されている植物は「文字情報」が中心です。しかし実際の試験では、「この葉の形の樹木は何か」という視覚的な問われ方もあります。

**おすすめは「樹木カード」の自作です。**

市販の単語カードに以下の情報を書いて、通勤中に毎日5枚ずつ覚えました。

- 樹木名（和名・カタカナ）
- 科名（例：ブナ科・バラ科・マメ科）
- 常緑・落葉の区別
- 針葉・広葉の区別
- 代表的な病害虫（例：サクラ＝てんぐ巣病・アメリカシロヒトリ）
- 剪定の適期（例：常緑樹は新芽が固まった6月・9月前後）

最初の1ヶ月で覚えるのは「頻出20種」だけで十分です。過去問を見ると、**サクラ・マツ・ケヤキ・クスノキ・ツバキ・カシ・スギ・ヒノキ・キンモクセイ・ウメ**あたりが繰り返し出題されています。頻出種から攻略していきましょう。

### 現場で「ながら学習」する

造園の仕事をしている方は、**現場そのものが教材**です。

木を植えるとき→「この樹木の科名は？常緑か落葉か？」
剪定するとき→「この病気の原因は？適切な対処法は？」
公園設備を確認するとき→「都市公園法でこの施設の面積基準は？」

こういった問いかけを習慣にするだけで、仕事中の時間がそのまま学習時間になります。私はこの「現場でながら学習」だけで、植物知識の約半分を身につけました。

### 法規の覚え方

都市公園法・労働安全衛生法・建設業法の重要数値は、まとめシートを作って毎朝確認する方法が最も効果的です。

| 頻出数値の例 | 内容 |
|------------|------|
| 街区公園の標準面積 | 0.25ha |
| 近隣公園の標準面積 | 2ha |
| 地区公園の標準面積 | 4ha |
| 総合公園の標準面積 | 10〜50ha |
| 高木の定義（建設工事公衆災害防止対策要綱） | 樹高3m以上 |
| 元請業者の下請総額の上限（特定建設業の要件） | 4,500万円以上（建築一式は7,000万円以上） |

これらは毎年必ずといっていいほど出題されます。A4用紙1枚にまとめて、昼休みや就寝前に確認するだけで自然に覚えられます。

## 全国建設研修センターの活用法

造園施工管理技士の試験を実施しているのは**一般財団法人 全国建設研修センター**です。

試験勉強以外でも、以下のようにセンターを積極的に活用することをおすすめします。

### 過去問の無料ダウンロード

全国建設研修センターのウェブサイトでは、過去数年分の試験問題（第一次検定）を無料でダウンロードできます。

**活用法：**
- 印刷して本試験と同じ形式で解く（時間を計って解くと緊張感が出る）
- 過去問集に載っていない最新年度の問題をカバーできる
- 無料なので、まず試験の形式・難易度を確認したい方にも最適

### 受験の手引きで出題範囲を確認する

センターが公開している「受験の手引き」には、出題科目・出題範囲・試験時間が明記されています。テキストを購入する前にこれを確認することで、「何をどこまで勉強すればいいか」の全体像が見えます。

### センター主催の講習会

年によってはセンターや関連機関が実施する対策講習会が開催されることがあります。独学が不安な方や、経験記述の添削を受けたい方は検討してみてください。私は受講しませんでしたが、「文章が得意でない方」には有効な選択肢だと思います。

## 買う前に確認！チェックリスト

テキスト・問題集を購入する前に、以下を確認してください。

- [ ] 最新年度（2〜3年以内）の問題が収録されているか
- [ ] 植物のカラー図版・写真が入っているか
- [ ] 解説が「答えだけ」でなく「なぜその答えか」まで書かれているか
- [ ] 都市公園法・労働安全衛生法の法規分野がカバーされているか
- [ ] 経験記述の記述例・文例が収録されているか（第二次検定対策）
- [ ] 全国建設研修センターの公式問題集と組み合わせて使えるか確認したか

## まとめ

造園施工管理技士のテキスト・過去問集は選択肢が少ない分、「正しい2冊を選んで徹底的にやりきる」ことが合格への最短ルートです。

| 目的 | 使用教材（私の実例） | 使い方 |
|------|------------|--------|
| 基礎知識の習得（メイン） | [例題で学ぶ!! 2級造園施工管理技術検定（弘文社）](https://www.amazon.co.jp/dp/4770328524)（新品） | 辞書として使う。過去問で間違えた箇所を読む |
| 補完・別角度からの理解 | 日建学院テキスト（中古） | メインで理解できない部分を補完 |
| 本試験形式の演習 | 全国建設研修センター公式問題集 | 試験1ヶ月前から演習 |
| 植物知識の補強 | 樹木カード自作＋現場でのながら学習 | 毎日5枚を継続 |

教材が少ない造園だからこそ、みんなが同じスタートラインに立っています。あとは「どれだけやり込んだか」だけです。

私は造園作業員として現場で木を触りながら、昼休みと通勤時間だけで2級造園・1級造園を一発合格しました。特別な才能も、予備校も必要ありませんでした。正しい教材を選んで、毎日コツコツ続けるだけです。

今日から始めてください。「テキストを買ったら、その日に過去問の1問目を解く」——これだけでいいです。
