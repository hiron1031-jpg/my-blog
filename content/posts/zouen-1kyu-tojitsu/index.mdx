1級造園は第一次が9月（残暑）／第二次が12月（真冬）という、気候差が最大の試験。さらに会場が土木より少なく地方は前泊覚悟という独自の悩みもあります。

この記事では、1級造園に独学・1発合格した筆者が **1級造園に特化した試験当日のチェックリスト** を解説します。

> 4資格共通の基本ルールは [施工管理技士 試験当日の持ち物・タイムスケジュール](/posts/shiken-tojitsu-jumbi) を参照。

## 1級造園 試験日と所要時間

| 検定 | 時期 | 所要時間 | 形式 |
|------|------|---------|------|
| 第一次検定 | **9月上旬（日）** | **4時間30分**（午前2.5h+午後2h） | マークシート4択 |
| 第二次検定 | **12月上旬（日）** | **2時間45分** | 記述式 |

> ⚠️ 試験日は年度で変動します。最新情報は [全国建設研修センター](https://www.jctc.jp/) の受験要項で必ず確認してください。
> 1級造園は土木と試験日が異なります（土木：7月／10月）。

## 【第一次検定 9月】残暑対策

9月の1級造園は残暑日（30度超）に当たることが多く、土木7月ほどではないが暑さ対策は必要。

### 第一次の追加持ち物と特有リスク

| 項目 | 理由 |
|------|------|
| **塩分タブレット / OS-1** | 残暑の熱中症対策 |
| 薄手の長袖 | 冷房ガンガンの会場対策 |
| ペットボトル飲料 | 移動中の水分補給 |
| 汗拭きシート | 着席前にリセット |
| 替えの靴下・インナー | 大量発汗で気分が悪くなる対策 |

9月特有のリスク：
- 台風直撃：9月は台風シーズン。**前日に進路確認**
- 残暑で熱中症：会場到着までで体力消耗
- 会場の冷房効きすぎ：上着がないと震える

---

## 【第二次検定 12月】真冬の防寒と記述対策

12月の1級造園は真冬の試験。寒さ対策と早朝の電車遅延（凍結・雪）に注意が必要です。

### 第二次の追加持ち物と試験中タイムスケジュール

| 項目 | 注意点 |
|------|------|
| **黒の油性ボールペン 3本以上** | フリクション不可 |
| 修正テープ | 訂正用 |
| カイロ（貼る・貼らない両方） | 真冬の試験会場は底冷え |
| ひざ掛け / ストール | 持ち込み可な会場が多い |
| マスク | 乾燥・風邪予防 |
| 使い慣れた経験記述ノート | 直前確認用 |

> ⚠️ **黒のボールペンは2本以上必須**。寒さでインク出にくい場合あり、試し書きしてから持参。

第二次の試験中タイムスケジュール：

| 時間帯 | やること |
|--------|---------|
| 0〜5分 | 全問題に目を通し、選択問題の取捨選択 |
| 5〜50分 | **経験記述（問題1）** を一気に書く |
| 50〜130分 | 専門記述（植栽・造園土工・修景・施工計画）を分野ごと |
| 130〜150分 | 見直し・誤字脱字・空欄ゼロ確認 |

> 💡 造園の経験記述は「植栽工事」「修景施設工事」など**造園特有の工事名**を必ず使うこと。土木表現だけだと造園経験と認められない可能性があります。

## 1級造園 会場の少なさ・前泊問題

1級造園は土木より受験者数が少なく、会場も限られます。主要会場は札幌・仙台・東京・名古屋・大阪・広島・高松・福岡・那覇など主要都市のみ。

地方在住者の前泊段取り：

| 項目 | 推奨 |
|------|------|
| ホテル予約 | 試験日が決まったら**即予約**（建設業界の研修と被りやすい） |
| ホテル選び | 会場から徒歩圏が最優先 |
| 朝食 | 朝食付きプランが時短に有利 |
| 持ち込み | カイロ・常備薬は前日のうちに準備 |
| 12月の特注意 | 雪国は新幹線遅延・空港閉鎖リスク → 前々日入りも検討 |

> 💡 9月の第一次は敬老の日前後の3連休に当たる年があり、12月の第二次は年末・忘年会シーズンでホテルが埋まりやすい。どちらも**早期予約が鉄則**。

## 1級造園 試験中の戦略・前日の過ごし方

### 第一次・第二次の戦略

**合格ラインは60%**。造園特有の植物名問題は捨て問題化してもOK（樹種暗記は時間対効果が低い）。1問2分以上悩んだら印をつけて次へ。

第二次検定は経験記述から書き始め（配点が最大）、「造園工事業」「植栽工」「修景施設工」のキーワードを必ず入れること。専門記述は植栽・支柱・養生あたりが頻出です。

### 前日の過ごし方

✅ やるべきこと：持ち物の最終確認（2回）・経路再確認（9月：台風／12月：雪）・要点ノート流し読み（30分）・**22時就寝**・前泊なら暖房・冷房早めにON

❌ やってはいけないこと：新しい問題集に手を出す・徹夜詰め込み・暴飲暴食・前泊先での飲み歩き

## よくある質問（FAQ）

### Q1. 12月の第二次検定はどれくらい寒いですか？対策は十分でしょうか？

会場によりますが、12月は底冷えすることがよくあります。**貼るカイロ2枚＋ひざ掛け＋重ね着**で対策するのが筆者推奨です。特に「書く」試験は手が冷えると文字が乱れます。薄手のインナーグローブも有効です。

### Q2. 地方在住でホテルが取れない場合はどうすればいいですか？

会場周辺ホテルは早期に埋まります。最悪、電車で30分以内の立地で妥協してください。12月は年末・忘年会シーズンと重なるため、受験票が届いた瞬間に**即予約**することが必要です。

### Q3. 第二次検定の経験記述で、土木の表現を使ってもいいですか？

使わない方が安全です。1級造園の経験記述では「植栽工事」「修景施設工」「造園土工」など**造園特有の工種名**を使うことが重要です。「土工」「護岸工」などの土木表現だけでは「造園工事の施工管理経験」として認められにくいです。
