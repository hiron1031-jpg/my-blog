2級造園は **11月の1日で第一次・第二次の両方** を受ける可能性がある資格です（同時受験を選んだ場合）。**朝〜夕方まで会場拘束**＋**初冬の寒さ**＋**会場が少ない地方在住者** という独自の負荷があります。

この記事では、2級造園に独学・1発合格した筆者が **2級造園に特化した試験当日のチェックリスト** を解説します。

> 4資格共通の基本ルールは [施工管理技士 試験当日の持ち物・タイムスケジュール](/posts/shiken-tojitsu-jumbi) を参照。

---

## 2級造園 試験日と所要時間

| 検定 | 時期 | 所要時間 | 形式 |
|------|------|---------|------|
| **第一次検定** | **11月上旬（日）午前** | 約2時間 | マークシート4択 |
| **第二次検定** | **11月上旬（日）午後** | 約2時間 | 記述式 |

> ⚠️ 2級造園は **第一次・第二次が同日**。同時受験を選んだ場合 **朝〜夕方まで会場拘束** になります。試験日は年度で変動するので [全国建設研修センター](https://www.jctc.jp/) で必ず確認。

## 2級造園 必須持ち物（共通）

| 項目 | 注意点 |
|------|------|
| **受験票** | 顔写真貼付済み |
| **写真付き本人確認書類** | **学生証だけでは不可**。運転免許 or マイナンバー |
| **HBの鉛筆 5本以上** | 第一次マークシート用 |
| **消しゴム 2個** | 1個は予備 |
| **腕時計（アナログ）** | スマートウォッチ不可 |

> 💡 19歳・大学生・植木屋見習いの受験者は **学生証・社員証のみ** で来てしまう失敗が多い。**写真付き公的身分証** を必ず持参。

## 【11月】初冬・1日通し受験の対策

11月の2級造園は **初冬の寒さ**＋**1日で2試験** の体力勝負。寒さで集中力が削がれないように、また昼食で午後の眠気を作らないように注意。

### 同日受験の追加持ち物

| 項目 | 理由 |
|------|------|
| **黒の油性ボールペン 3本以上** | **第二次の記述用**（フリクション不可） |
| **下書き用シャーペン** | 問題用紙余白で下書き |
| **昼食（おにぎり等）** | 会場周辺コンビニは混雑 |
| **チョコ / ラムネ** | 午後の集中力ブースト |
| **カイロ（貼る・貼らない両方）** | 11月の会場は底冷え |
| **ひざ掛け / ストール** | 持ち込み可な会場が多い |
| **マスク** | 乾燥・風邪予防 |
| **使い慣れた経験記述ノート** | 昼休みの直前確認用 |

> ⚠️ 昼食は **重い丼物・ラーメンは避ける**。午後の眠気で第二次が壊滅します。**おにぎり2個＋温かい味噌汁** くらいが筆者推奨。

## 2級造園 同日通し受験のタイムスケジュール（モデル）

| 時刻 | やること |
|------|---------|
| 6:30 | 起床・朝食 |
| 7:30 | 自宅出発（路面凍結 or 落ち葉スリップ注意） |
| 9:00 | 会場着・トイレ・カイロ貼る |
| 9:00〜9:30 | 要点ノート確認 |
| 9:45頃 | **第一次検定 開始** |
| 11:45頃 | 第一次終了 |
| 12:00〜13:00 | **昼食・トイレ・経験記述ノート確認** |
| 13:30頃 | **第二次検定 開始** |
| 15:30頃 | 第二次終了 |

> ⚠️ 試験開始時刻は **受験票記載の時刻** が正です。年度・会場で異なります。

### 昼休みの過ごし方が合否を分ける

第一次の出来が悪くても **絶対に引きずらない**。昼休み60分の使い方：

1. **トイレ**（午後の途中退室は集中力が切れる）
2. **昼食を10〜15分で軽く**
3. **経験記述ノートをペラペラ流し読み**（30分）
4. **5分仮眠**（机に伏せる程度）

## 2級造園 試験中の戦略

### 第一次検定（マークシート4択）

- **60%（合格ライン）** を確保すれば良い
- **植物名問題（樹種・草本）** は捨て問題化してもOK
- 1問2分以上悩んだら **印をつけて次へ**
- マークシートは10問ごとに **番号ズレ確認**

### 第二次検定（記述）

2級造園の経験記述は1級ほど高度な役職表現は不要ですが、**「植栽工」「造園土工」「修景施設工」** の用語は必ず使うこと。

- **経験記述から書く**（配点が最大）
- **「現場代理人補佐」「主任技術者補助」「工事担当者」** など2級レベルの役職でOK
- **空欄は作らない**（部分点狙い）
- 専門記述は **植栽・支柱・養生・剪定** あたりが頻出

> 💡 **「庭木の剪定・除草」だけの経験では受験資格が認められない** ように、記述でも「造園**工事**」のキーワードを必ず入れる。

## 2級造園 会場の少なさ・前泊問題

2級造園も土木より **受験者数が少なく、会場も限られます**。

### 主要会場のみ

- 札幌・仙台・東京・名古屋・大阪・広島・高松・福岡・那覇 など主要都市のみ

### 地方在住者の前泊段取り

| 項目 | 推奨 |
|------|------|
| ホテル予約 | 試験日が決まったら **即予約** |
| ホテル選び | 会場から **徒歩圏** が最優先 |
| 朝食 | **朝食付きプラン** が時短に有利 |
| 11月の特注意 | 北日本は **初雪・路面凍結** リスクあり |

> 💡 11月は **3連休（文化の日付近）** に当たる年があり、ホテル料金が高騰します。早期予約が鉄則。

## 19歳・植木屋見習い受験者へのアドバイス

2級造園は **19歳以上で実務経験ゼロでも第一次検定が受けられる** ため、若手・植木屋の見習いの受験が増えています。

### 若手受験者が注意すること

1. **本人確認は学生証・社員証だけでは不可**。**写真付き公的身分証** を持参
2. **作業着で来場OK**（造園会社員は作業着のまま来る人も）
3. **休憩時間にスマホ禁止**（カバンに入れて電源OFF）
4. **保護者・先輩同伴NGエリアあり**
5. 周囲は **30〜60代の造園職人が多い** → 物怖じしない

## 2級造園 前日の過ごし方

### ✅ やるべきこと

1. **持ち物の最終確認**（チェックリストで2回）
2. **経路再確認**（11月：初雪・凍結）
3. **要点ノート / 経験記述テンプレ** の流し読み（30分）
4. **22時就寝**
5. **前泊なら部屋の暖房** 早めにON

### ❌ やってはいけないこと

1. **新しい問題集** に手を出す
2. **徹夜詰め込み**
3. **暴飲暴食**
4. **前泊先での飲み歩き**
5. **会社の忘年会** → 当日に頭が回らない

## まとめ

| 項目 | 2級造園のポイント |
|------|------------------|
| 試験日 | **11月上旬（第一次・第二次 同日）** |
| 通し受験 | 朝〜夕方まで会場拘束 → 昼食・体力配分が肝 |
| 必須5点 | 受験票・**写真付き身分証**・鉛筆・消しゴム・時計 |
| 寒さ対策 | カイロ・ひざ掛け・マスク |
| 第二次 | **黒のボールペン2本以上**・経験記述ノート |
