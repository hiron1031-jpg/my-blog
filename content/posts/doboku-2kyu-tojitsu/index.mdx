2級土木は **前期（6月・第一次のみ）／後期（10月・第一次＋第二次同日）** という年2回制度。**1日で第一次・第二次両方** を受ける後期は、1級とは違う体力配分が必要です。

この記事では、2級土木に独学・1発合格した筆者が **2級土木に特化した試験当日のチェックリスト** を解説します。

> 4資格共通の基本ルールは [施工管理技士 試験当日の持ち物・タイムスケジュール](/posts/shiken-tojitsu-jumbi) を参照。

---

## 2級土木 試験日と所要時間

| 検定 | 時期 | 所要時間 | 形式 |
|------|------|---------|------|
| **前期 第一次のみ** | **6月上旬（日）** | 約2時間 | マークシート4択 |
| **後期 第一次** | **10月下旬（日）午前** | 約2時間 | マークシート4択 |
| **後期 第二次** | **10月下旬（日）午後** | 約2時間 | 記述式 |

> ⚠️ **後期は午前・午後の通し受験** が選択可能。同日に第一次・第二次の両方を受ける場合、**朝〜夕方まで会場拘束** になります。

## 2級土木 必須持ち物（共通）

| 項目 | 注意点 |
|------|------|
| **受験票** | 顔写真貼付済み |
| **写真付き本人確認書類** | **学生証だけでは不可**。運転免許 or マイナンバー |
| **HBの鉛筆 5本以上** | マークシート用 |
| **消しゴム 2個** | 1個は予備 |
| **腕時計（アナログ）** | スマートウォッチ不可 |

> 💡 19歳・大学生の受験者は **学生証のみ** で来てしまう失敗が多い。**写真付き公的身分証** を忘れずに。

---

## 【前期 6月】梅雨対策が最重要

6月の2級土木は **梅雨真っ只中**。ほぼ毎年、どこかの会場で **豪雨・電車遅延** が起きています。

### 前期の追加持ち物

| 項目 | 理由 |
|------|------|
| **折りたたみ傘 + ビニール袋** | 受験票が濡れると印字が滲む |
| **替えの靴下** | 革靴・スニーカーの浸水対策 |
| **タオル** | 髪・服の水滴を拭く |
| **クリアファイル** | 受験票の防水 |
| **薄手の長袖** | 冷房対策 |

### 6月特有のリスク

- **電車遅延**：豪雨で1時間以上の遅延もあり得る → **2時間前出発**
- **道に迷う**：傘で視界が狭い・地図が見にくい → **前日に経路確認**
- **会場の冷房効きすぎ**：濡れた服で震える → 上着・タオル必須

---

## 【後期 10月】1日通し受験の体力配分

後期は **第一次（午前）＋第二次（午後）** が可能。**朝9時〜夕方17時** くらいまで会場にいるイメージです。

### 後期の追加持ち物

| 項目 | 理由 |
|------|------|
| **黒の油性ボールペン 3本以上** | **第二次の記述用**（フリクション不可） |
| **下書き用シャーペン** | 問題用紙余白で下書き |
| **昼食（おにぎり等）** | 午後の試験に備えて軽め |
| **チョコ / ラムネ** | 午後の集中力ブースト |
| **カイロ** | 10月の会場は冷えることあり |

> ⚠️ 昼食は **重い丼物・ラーメンは避ける**。午後の眠気で第二次が壊滅します。

### 後期 通し受験のタイムスケジュール（モデル）

| 時刻 | やること |
|------|---------|
| 6:30 | 起床・朝食 |
| 7:30 | 自宅出発 |
| 9:00 | 会場着・トイレ |
| 9:45 | **第一次検定 開始** |
| 11:45 | 第一次終了 |
| 12:00〜13:00 | 昼食・トイレ・経験記述ノート確認 |
| 13:30 | **第二次検定 開始** |
| 15:30 | 第二次終了 |

> ⚠️ 試験開始時刻は **受験票記載の時刻** が正です。年度・会場で異なります。

## 2級土木 試験中の戦略

### 第一次検定（マークシート4択）

- **60%（合格ライン）** を確保すれば良い
- **必須問題と選択問題** の取捨選択を最初に確定
- 1問2分以上悩んだら **印をつけて次へ**
- マークシートは10問ごとに **番号ズレ確認**

### 第二次検定（記述）

2級土木の第二次は1級より **軽い記述量** ですが、経験記述は同じくらい重要です。

- **経験記述から書く**（配点が最大）
- **黒のボールペンで丁寧に**
- **空欄は作らない**（部分点狙い）
- 専門記述は **キーワードを必ず入れる**

---

## 19歳・学生受験者へのアドバイス

2級土木は **19歳以上で実務経験ゼロでも第一次検定が受けられる** ため、大学生・専門学校生の受験が増えています。

### 学生受験者が注意すること

1. **本人確認は学生証だけでは不可**。**写真付き公的身分証** を持参
2. **休憩時間にスマホ禁止**（カバンに入れて電源OFF）
3. **試験当日は私服でOK**（スーツ不要）
4. **保護者同伴NGエリアあり**（会場ロビーまで）
5. 周囲は **30〜50代のおじさんが多い** → 物怖じしない

---

## 2級土木 前日の過ごし方

### ✅ やるべきこと

1. **持ち物チェックリスト**（2回確認）
2. **経路確認**（梅雨/秋の天気予報チェック）
3. **要点ノート流し読み**（30分）
4. **22時就寝**

### ❌ やってはいけないこと

1. **新しい問題集** に手を出す
2. **徹夜詰め込み**
3. **暴飲暴食**（特に6月の生もの・刺身は危険）
4. **大学のサークル飲み会** → 当日に頭が回らない

## まとめ

| 項目 | 2級土木のポイント |
|------|------------------|
| 前期 | **6月上旬（梅雨）** 傘・替え靴下必須 |
| 後期 | **10月下旬（第一次・第二次通し）** 昼食・ボールペン必須 |
| 必須5点 | 受験票・**写真付き身分証**・鉛筆・消しゴム・時計 |
| 学生注意 | 学生証のみは不可。免許 or マイナンバー必須 |
| 通し受験 | 昼食は軽め（眠気防止） |
| 移動 | 会場到着 **45分前** 目標 |
| 前日 | 22時就寝・天気予報チェック |

2級土木は **若手・学生のキャリア入口** として最適な資格です。後期の通し受験は体力勝負——朝食・昼食の段取り次第で第二次の出来が大きく変わります。
