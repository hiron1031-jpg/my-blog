1級造園施工管理技士の **受験申込み** は、土木に比べて **情報が少ない** のが最大の悩みどころです。「指定学科に造園が含まれるのか？」「植栽工事だけの実務経験で認められるのか？」など、調べてもまとまった情報が見つからない人が多いはずです。

この記事では、1級造園に独学・1発合格した筆者が、**1級造園に特化した受験申込み手順** と **造園独自の実務経験の書き方** を解説します。

> 4資格まとめての比較は [施工管理技士 受験申込み完全ガイド](/posts/shikaku-moshikomi-flow) を参照。

## 1級造園 受験申込みの流れ

| ステップ | 時期（目安） | やること |
|---------|-----------|---------|
| 1. 受験要項確認 | 5月頃 | 全国建設研修センターHPで公表 |
| 2. 申込み | 5月中旬〜6月上旬 | インターネット推奨 |
| 3. 実務経験証明書準備 | 申込みと同時 | 会社印・所属長の証明 |
| 4. 受験票受領 | 8月下旬 | 試験会場の確定 |
| 5. **第一次検定** | **9月上旬** | 結果は10月発表 |
| 6. **第二次検定** | **12月上旬** | 結果は3月発表 |
| 7. 合格証明書 | 3月以降 | 国土交通大臣名の証明書 |

> ⚠️ 試験日程は年度で変動します。最新情報は [一般財団法人 全国建設研修センター](https://www.jctc.jp/) で必ず確認してください。
> 1級造園は **試験日が土木と異なる**（土木は7月／造園は9月）ので、混同しないよう注意してください。

## 1級造園 受験資格

### 第一次検定：19歳以上で受験可能

実務経験不要で、19歳以上であれば誰でも受験可能。

### 第二次検定：学歴別の実務経験要件

| 学歴区分 | 指定学科卒 | 指定学科以外卒 |
|---------|----------|------------|
| 大学 | **3年以上** | 4.5年以上 |
| 短大・高専 | 5年以上 | 7.5年以上 |
| 高校 | 10年以上 | 11.5年以上 |
| その他（中卒等） | 15年以上 | 15年以上 |

> ※ 令和6年度から制度改定があり、第一次検定合格後の実務経験ルートも追加されました。最新の要件は必ず公式要項を確認してください。

### 1級造園の「指定学科」（土木と異なる）

造園の指定学科は **土木と少し範囲が異なります**。

- **造園学科 / 造園科**
- **農学科 / 園芸学科 / 園芸農学科**
- **林学科 / 林業学科**
- **農業土木学科**
- **土木工学科**（造園にも含まれる）
- **緑地計画学科**

> ⚠️ 「建築学科」は **造園では指定学科に該当しません**（土木では該当）。逆に「林学・農学・園芸」は造園特有の指定学科です。自分の卒業学科がどっちに該当するか、卒業証明書で正確に確認してください。

## 認められる実務経験（造園編）

造園の実務経験として認められる工事範囲は **造園工事業に関する工事** に限られます。

### ✅ 認められる造園工事

- **植栽工事**（街路樹・公園緑化・庭園植栽・屋上緑化）
- **移植工事**（既存樹木の保存移植）
- **造園土工**（築山・整地・客土）
- **石材工**（石組み・飛石・延段）
- **竹垣工 / 生垣工**
- **修景施設工**（園路・広場・水景）
- **遊戯施設工**（公園遊具設置）
- **排水工**（造園排水）
- **管理施設工**（造園に関する境界フェンス等）

### ❌ 認められない業務

- **庭木の剪定・除草のみ**（管理業務であり工事ではない）
- **農業従事**（果樹園経営など）
- **花卉栽培・販売**
- **造園設計のみ**（施工管理を伴わない）
- **建築外構の土木工事**（造園でなく土木）

> 💡 「庭木の剪定・除草」などの **造園維持管理業務** は実務経験に **含まれません**。あくまで「造園**工事**」の施工管理業務が対象です。

## 【最重要】1級造園の実務経験証明書 記入例

### ✅ 良い記入例（1級造園）

| 期間 | 工事名・発注者 | 職務内容 |
|------|--------------|---------|
| 令和2年4月〜令和3年3月 | ○○公園植栽工事（○○市） | **現場代理人補佐** として、植栽工・園路工の **施工管理**（工程・品質・安全）を担当 |
| 令和3年4月〜令和4年3月 | ○○団地外構緑化工事（○○県住宅供給公社） | **主任技術者** として、植栽・支柱・客土工事の施工管理を統括 |
| 令和4年4月〜令和5年9月 | ○○庭園修景工事（○○市文化財課） | **現場代理人** として、石組み・園路・植栽工の施工計画・品質管理を実施 |

### 1級造園の職務内容で必ず使うキーワード

- **施工管理**（工程・品質・安全・原価）
- **現場代理人** / **現場代理人補佐**
- **主任技術者** / **監理技術者補佐**
- **植栽工** / **造園土工** / **修景施設工**
- **施工計画の立案**
- **出来形管理 / 品質管理**

### 造園特有の書き方のコツ

1級造園の職務内容では、**「造園工事業の主たる工事」** であることを明確に書く必要があります。

**✅ 効果的な表現：**
- 「**造園工事業** として、植栽工・修景施設工の施工管理を担当」
- 「**公園緑化工事** において、樹木支柱・客土工事を統括」

**❌ 曖昧な表現：**
- 「外構工事の施工」 → 造園か土木か不明
- 「緑化作業」 → 工事か作業員か不明

## 1級造園 受験申込みでよくある失敗

### 失敗1：庭木剪定・除草の経験を申告

**結果**：管理業務扱いで不受理。
**対策**：剪定・除草は実務経験に含まれません。植栽工事・移植工事など **「工事」** の経験のみを記載してください。

### 失敗2：農学科出身者の指定学科確認漏れ

**対策**：農学科・園芸学科・林学科は **造園の指定学科に該当** します。卒業証明書を取り寄せて学科名を確認。

### 失敗3：建築外構工事を造園経験として申告

**対策**：建築工事に附帯する外構の土木部分は **土木の経験** であり、造園ではありません。植栽が主体の工事のみ造園経験として認められます。

### 失敗4：試験日を土木と混同

**対策**：1級造園は **9月（第一次）／12月（第二次）**。土木は **7月／10月**。同年に両方受けることも可能です。

## 1級造園の受験料・受験会場

### 受験料

- **第一次検定**：14,400円程度
- **第二次検定**：14,400円程度
- **第一次・第二次同時受験**：28,800円程度

> ※ 金額は年度で変動。最新は公式要項を確認。土木より受験料が高めの傾向。

### 試験会場（土木より会場数が少ない）

1級造園は **土木より受験者数が少ない** ため、会場も限られます。
- 札幌・仙台・東京・名古屋・大阪・広島・高松・福岡・那覇 など主要都市のみ

地方在住者は **試験会場まで遠征が必要** な場合があるので、早めの会場選択と宿泊予約がおすすめです。

## まとめ

| 項目 | 1級造園のポイント |
|------|------------------|
| 第一次検定 | 19歳以上で受験可（実務経験不要） |
| 第二次検定の実務経験 | 大卒・指定学科で **3年以上** |
| 指定学科 | 造園・農学・園芸・林学・土木 等 |
| 試験日 | 第一次：**9月**／第二次：**12月** |
| 認められる工事 | 植栽・移植・修景・園路・石材工 等 |
| 認められない業務 | 庭木剪定・除草・農業・花卉販売 |
| 必須キーワード | 造園工事業・施工管理・植栽工 |
| 会場数 | 土木より少ない（地方は遠征覚悟） |
