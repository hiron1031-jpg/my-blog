## 持ち物チェックリスト【絶対忘れないもの】

### ✅ 必須5点（これがないと受験不可）

| 項目 | 注意点 |
|------|------|
| **受験票** | 顔写真貼付済み・印刷済みのもの。スマホ画面は不可 |
| 写真付き本人確認書類 | 運転免許証・マイナンバーカード・パスポートのいずれか |
| HBの鉛筆 5本以上 | シャーペンは不可の場合あり。HB鉛筆推奨 |
| **消しゴム 2個** | 1個は必ず予備。プラスチック消しゴム推奨 |
| 時計（アナログ） | 会場に時計がない場合あり。スマホ・スマートウォッチは不可 |

> ⚠️ **本人確認書類は健康保険証だけでは不可** の場合があります。必ず写真付きを準備してください。

### ✅ 推奨10点（あると安心）

| 項目 | なぜ必要か |
|------|---------|
| 鉛筆削り | 鉛筆の芯が折れた時の保険 |
| マークシート用消しゴム | 細かい修正がしやすい |
| お弁当・飲み物 | 会場周辺にコンビニがない場合あり |
| 上着 / ひざ掛け | 冷房対策（夏でも会場は寒いことが多い） |
| 耳栓 | 周囲の音が気になる人は持参 |
| 目薬 | 長時間集中で目が乾く |
| 常備薬 | 頭痛薬・胃薬・絆創膏 |
| マスク | 体調管理＆周囲への配慮 |
| 現金（小銭） | 自販機・コインロッカー用 |
| ハンカチ・ティッシュ | 当然 |

### ✅ 第二次検定 追加持ち物

| 項目 | 注意点 |
|------|------|
| **黒のボールペン** | 第二次検定の記述は消せないペンで記入。消せるボールペン（フリクション）は不可 |
| 修正テープ | 訂正は二重線＋訂正印が基本だが、修正テープも認められる場合あり（要・要項確認） |
| 下書き用シャーペン | 解答用紙の下書き欄や問題用紙の余白用 |

> 💡 記述は消せないペンが原則です。第二次検定でフリクションを使うと無効になる可能性があるため、必ず黒の油性ボールペンを2本以上持参してください。

## 試験当日のタイムスケジュール（1級土木 第一次検定の例）

| 時刻 | やること |
|------|---------|
| 6:30 | 起床・朝食（消化のいいもの） |
| 7:30 | 自宅出発（電車遅延も想定して会場まで余裕を1時間） |
| 8:30 | 会場最寄駅到着・コンビニで飲み物補充 |
| 9:00 | 会場到着・席確認・トイレ |
| 9:00〜9:30 | 直前確認（暗記カード・要点ノートのみ） |
| 9:30 | 試験官入室・諸注意 |
| 9:45 | 第一次検定 **午前の部** 開始（2時間30分） |
| 12:15 | 午前の部 終了 |
| 12:15〜13:30 | 昼休憩（**会場内で待機推奨**） |
| 13:35 | **午後の部** 開始（2時間） |
| 15:35 | 午後の部 終了 |

> ⚠️ 1級土木 第一次検定は **午前2.5時間＋午後2時間の計4時間30分** の長丁場（R6制度改定後）。試験開始時刻・所要時間は資格・年度で異なります。**受験票記載の時刻** を必ず確認してください。

### 移動時間の鉄則：「最低1時間の余裕」を持つ

電車遅延・道に迷う・トイレ並びすぎ——当日のトラブルは想像以上に多いです。**会場到着は試験開始の45分前**を目標にしてください。

特に首都圏・大阪などの大規模会場は最寄駅から徒歩15分以上あることも珍しくありません。事前にGoogleマップで経路を確認しておくこと。

## 試験中の戦略

### 第一次検定（マークシート4択）

**時間配分の黄金ルール：**

| 周回 | やること |
|------|---------|
| 1周目（60〜70%の時間） | 全問解く。確実な問題と「迷う問題」をマーク |
| 2周目（20%の時間） | 迷う問題を再検討。3択・2択まで絞る |
| 3周目（10%の時間） | マーク漏れ・マークズレを最終チェック |

ポイント：
- 1問に2分以上悩まない。迷ったら印をつけて次へ進む
- **マークシートは1問ズレると全滅。**10問ごとに番号確認
- 60%が合格ラインなので、全問正解は不要。捨て問題も戦略

### 第二次検定（記述式）

**時間配分：**

| 時間帯 | やること |
|--------|---------|
| 0〜10分 | 全問題に目を通し、選択問題の取捨選択を決定 |
| 10〜60分 | 経験記述（問題1）を一気に書く |
| 60〜140分 | 専門記述を分野ごとに記述 |
| 140〜160分 | 見直し・誤字脱字・空欄ゼロを確認 |

> 詳細は [1級土木 第二次検定 記述問題対策](/posts/doboku-1kyu-2ji-kijutsu) を参照。

## 前日の過ごし方【新しい問題に手を出さない】

### ✅ やるべきこと

1. **持ち物の準備**（チェックリストで2回確認）
2. 会場までの経路再確認（電車の時刻表・乗り換え）
3. 要点ノート・暗記カードの最終確認（30分以内）
4. 早めの就寝（22〜23時には就寝）
5. 当日の服装準備（重ね着で温度調節できる服）

### ❌ やってはいけないこと

1. 新しい問題集に手を出す → 自信を失うだけ
2. 徹夜で詰め込み → 当日に集中力が続かない
3. 大量飲酒 → 当日の頭が回らない
4. 激辛・脂っこい食事 → 当日にお腹を壊すリスク
5. SNSで情報収集 → 不安が増えるだけ

筆者の前日ルーティンは「18時に勉強終了 → 風呂 → 軽い夕食 → **22時就寝**」でした。前日に焦って詰め込んでも、当日のパフォーマンスは下がるだけです。

## 試験当日の朝の鉄則

### 朝食は「いつも通り」のメニュー

試験当日に特別なものを食べる必要はありません。むしろ普段食べないメニューはお腹を壊すリスクがあります。

筆者は試験日もいつものパン・コーヒー・卵でした。

### コーヒー・カフェインは控えめに

試験中のトイレが心配な人は、**朝のコーヒーは1杯まで** がおすすめ。

### 余裕がない時の最終確認は「要点ノート」のみ

新しい教材を開かない。いつも見ている自分の手書きノートを流し読みする程度で十分です。

## よくある質問（FAQ）

### Q1. 試験会場には何分前に到着すべきですか？

**試験開始45分前**を目標にしてください。会場によっては入場できる時間が決まっており、早すぎると待機スペースで時間を過ごすことになります。ギリギリ到着はメンタル的なダメージが大きいため、余裕を持った出発を推奨します。

### Q2. 第二次検定に鉛筆は使えますか？

経験記述・記述問題には黒の油性ボールペンを使用します。**フリクション（消えるボールペン）は会場の温度変化でインクが消える可能性があるため不可**です。マークシート部分は鉛筆またはシャープペンシルを使用します。

### Q3. 試験中にトイレに行けますか？

一般的に試験開始30分後〜終了10分前の間、試験官に申告してトイレに行けます。ただし試験要項で確認することを推奨します。当日は開始前にトイレを済ませておくのがベストです。

### Q4. 前日に徹夜で勉強するのは効果がありますか？

逆効果です。睡眠不足は記憶の定着と集中力を著しく低下させます。**前日は22時には就寝**し、十分な睡眠を取ることが最後の合格対策です。前日に新しい問題に手を出すのも不安を増やすだけなので避けてください。
