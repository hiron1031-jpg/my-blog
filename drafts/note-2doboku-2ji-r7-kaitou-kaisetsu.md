# 【無料公開】問題2

## 問題2（必須）：品質管理の穴埋め

### 問題の概要

建設工事における品質管理に関する文章の（イ）〜（ホ）に、語句又は数値群から適切なものを選んで記入する問題です。

### 想定解答

- **（イ）コーン指数**：トラフィカビリティーは、円錐型の先端を地盤に押し込むコーンペネトロメータで測定した「コーン指数 qc」で判断します。
- **（ロ）地盤反力**：平板載荷試験は、荷重と沈下量の関係から「地盤反力係数 K値」や極限支持力を調べる試験です。
- **（ハ）水セメント比**：コンクリート配合で「65%以下、かつ所要の強度・耐久性を満たす最小値」に設定するのは水セメント比（W/C）です。
- **（ニ）25**：日平均気温が「25℃」を超えると想定される場合は暑中コンクリートとして施工します（寒中コンクリートは日平均気温4℃以下が目安。セットで暗記）。
- **（ホ）CBR**：路床の強さはCBR試験で判定します。舗装厚さ設計の基礎となる値です。

### 採点ポイント・よくあるNG

- （イ）を「塑性指数」と書くNGが多発します。「トラフィカビリティー＝建設機械の走行性＝**コーン指数**」と機械的に結びつけてOK。
- （ニ）は15・20・25で迷わせる出題。**暑中=25℃超、寒中=4℃以下**は2級で最頻出の数値です。
- この問題は語句群から選ぶ形式なので、**漢字ミスの心配がない=確実に5点源**。過去問の穴埋め頻出語句（コーン指数・水セメント比・CBR・スランプ・塩化物）は毎年のように再登場します。

# 問題3（必須）：工程表の特徴（記述）

### 問題の概要

①バーチャート、②グラフ式工程表、③出来高累計曲線 から**2つ選び**、番号とその特徴を記述する問題です。

### 想定解答（3つとも掲載・このうち2つを記述すればOK）

**① バーチャート**
縦軸に工種（作業名）、横軸に日数（暦日）をとり、各作業の開始・終了時期と所要日数を横棒で表した工程表。作成が容易で各作業の期間が直感的に分かる反面、作業相互の関連や、工期に影響する重点作業がつかみにくい。

**② グラフ式工程表**
縦軸に出来高比率（%）、横軸に日数をとり、工種ごとの進捗を斜めの線で表した工程表。予定と実績の対比が視覚的に分かり、各作業の進捗状況が把握しやすいが、作業相互の関連は明確にできない。

**③ 出来高累計曲線**
縦軸に工事全体の出来高累計（%）、横軸に工期（日数）をとり、進捗をS字カーブで表した曲線。工事全体の進捗ペースが一目で分かり、許容範囲を示す上方・下方限界線（バナナ曲線）と組み合わせて全体管理に用いる。個々の作業の管理はできない。

### 採点ポイント・よくあるNG

- 「縦軸に何をとるか・横軸に何をとるか」を必ず入れる。これが入っていない説明は減点対象になりやすい。
- 長所だけでなく**短所（〜は分かりにくい）まで書くと安全**。「特徴について記述」は長所＋短所のセットが定石です。
- ネットワーク式工程表の説明を書いてしまう取り違えに注意（R7の選択肢にはありません）。

# 問題4（必須）：盛土の施工（穴埋め）

### 問題の概要

盛土の施工に関する文章の（イ）〜（ホ）に、語句群から適切なものを選んで記入する問題です。

### 想定解答

- **（イ）地耐力**：基礎地盤が軟弱で「地耐力」が不足する場合は、置換・表層処理・載荷・脱水・締固め・固結等で地盤改良します。
- **（ロ）せん断**：盛土材料は「せん断強度」が大きいものを用います。
- **（ハ）圧縮性**：同じく「圧縮性」が小さい（沈下しにくい）ものが適します。
- **（ニ）ICT**：電子情報を活用した効率的・高精度な施工は「ICT土工」。i-Construction関連で近年の超頻出テーマです。
- **（ホ）3次元**：起工測量や設計形状を「3次元データ」に変換して活用します（マシンガイダンス・マシンコントロール等）。

### 採点ポイント・よくあるNG

- 盛土材料の条件は「**せん断強度が大きく、圧縮性が小さく、施工が容易、雨水等で軟化・膨張しない**」がワンセット。どこが空欄になっても答えられるように。
- ICT・3次元・UAVなどの**i-Construction系用語はR6以降の新傾向**。来年度以降も狙われます。「2次元」「4次元」のひっかけに注意。

# 問題5（必須）：コンクリート用語の説明（記述）

### 問題の概要

①スペーサ、②コールドジョイント、③ブリーディング、④ワーカビリティー から**2つ選び**、番号と用語の説明を記述する問題です。

### 想定解答（4つとも掲載・このうち2つを記述すればOK）

**① スペーサ**
鉄筋やPC鋼材と型枠との間隔を正しく保持し、所定のかぶりを確保するために用いる部材。モルタル製・コンクリート製・鋼製等があり、型枠に接する部分には原則としてモルタル製又はコンクリート製を使用する。

**② コールドジョイント**
先に打ち込んだコンクリートと後から打ち重ねたコンクリートが、打重ね時間の超過等により一体化せず、打継ぎ部に生じる不連続面。強度・水密性・耐久性の低下の原因となる。

**③ ブリーディング**
コンクリートの打込み・締固め後、骨材やセメント等の沈降に伴って、練混ぜ水の一部が遊離して表面に上昇する現象。表面に形成されるレイタンスは打継ぎ前に除去する必要がある。

**④ ワーカビリティー**
材料分離を生じることなく、運搬・打込み・締固め・仕上げ等の作業が容易にできる程度を表すフレッシュコンクリートの性質。

### 採点ポイント・よくあるNG

- 「何のための／何が原因の」まで書くと得点が安定します（例：スペーサ→**かぶり確保のため**、コールドジョイント→**打重ね時間超過が原因**）。
- ブリーディングとレイタンスの取り違えに注意。**ブリーディング＝水が上がる現象、レイタンス＝その結果表面にできる脆弱な物質**です。
- この4用語はクイズ・一次検定でも頻出。[過去問チャレンジ（無料Webクイズ）](https://doboku-torisetsu.com/quiz)で反復できます。

# 問題6（選択(1)）：盛土の締固め管理（穴埋め）

### 問題の概要

盛土の締固め管理方法に関する文章の（イ）〜（ホ）に、語句群から適切なものを選んで記入する問題です。※問題6・7はどちらか1問のみ解答。

### 想定解答

- **（イ）強度**：締固めにより土の空気間隙を少なくし、土の構造物として必要な「強度」特性を得ます。
- **（ロ）抵抗**：完成後の盛土に求められる変形「抵抗」及び圧縮「抵抗」を確保します。
- **（ハ）含水**：締固めの効果や特性は、土の種類や「含水」の状態（含水比）・施工方法によって大きく変化します。
- **（ニ）仕様書**：品質規定方式は、必要な品質を「仕様書」に明示する方式です。
- **（ホ）施工者**：締固め工法そのものは「施工者」に任せ、締固め度等で品質を規定します。

### 採点ポイント・よくあるNG

- **品質規定方式と工法規定方式の対比**が最重要ポイント。
  - 品質規定方式＝品質（締固め度等）を仕様書で規定し、**工法は施工者に任せる**
  - 工法規定方式＝機種・敷均し厚さ・転圧回数等の**工法そのものを規定する**
- （ニ）（ホ）で「契約書」「発注者」を選ぶNGが定番のひっかけです。

# 問題7（選択(1)）：レディーミクストコンクリートの受入れ検査（穴埋め）

### 問題の概要

JIS A 5308 レディーミクストコンクリートの受入れ検査に関する文章の（イ）〜（ホ）に、語句又は数値群から適切なものを選んで記入する問題です。

### 想定解答

- **（イ）荷卸**：受入れ検査は「荷卸」時（荷卸し地点）に行います。
- **（ロ）スランプ**：規定値12cmに対し許容差±2.5cmなのは「スランプ」です（スランプ8cm以上18cm以下の場合±2.5cm）。
- **（ハ）4.5**：普通コンクリートの空気量は「4.5%」、許容差±1.5%。
- **（ニ）塩化物**：塩化物含有量は塩化物イオン量として「0.3kg/m³以下」（承認を受けた場合は0.6kg/m³以下）。
- **（ホ）3**：圧縮強度の1回の試験結果は、任意の1運搬車から採取した「3本」の供試体の平均値です。

### 採点ポイント・よくあるNG

- **受入れ検査の数値セットは2級二次の超頻出**です。丸ごと暗記推奨：
  - スランプ8〜18cm → 許容差 **±2.5cm**
  - 空気量：普通 **4.5%** ／軽量 5.0%（許容差 ±1.5%）
  - 塩化物イオン **0.3kg/m³以下**
  - 強度：**3本の平均**が呼び強度の85%以上、かつ各回が呼び強度の値以上
- （ハ）に「3.5」「5.5」を置くひっかけが定番。「普通=4.5」と即答できるレベルまで反復を。

# 問題8（選択(2)）：移動式クレーン作業の留意点（記述）

### 問題の概要

移動式クレーンを用いた作業で、クレーン等安全規則に定められた**事業者が実施すべき施工上の留意点を2つ**記述する問題です（玉掛け作業は除く）。※問題8・9はどちらか1問のみ解答。

### 想定解答（このうち2つを記述すればOK）

- 移動式クレーンの**定格荷重を超える荷重**をかけて使用してはならない。
- 上部旋回体との**接触のおそれがある箇所や、つり荷の下に労働者を立ち入らせない**（立入禁止措置を講じる）。
- **アウトリガーは最大限に張り出し**、地盤が軟弱な場合は鉄板等を敷いて地盤の支持力を確保する。
- あらかじめ移動式クレーンの**作業方法・転倒防止方法・労働者の配置と指揮系統**を定める。
- **一定の合図を定め、合図を行う者を指名**して、その者に合図を行わせる。
- **強風のため危険が予想されるときは作業を中止**する。
- 移動式クレーン**検査証を備え付け**、定格荷重を常時知ることができるよう表示する。
- 作業開始前に、**巻過防止装置・ブレーキ・クラッチ・コントローラーの機能を点検**する。

### 採点ポイント・よくあるNG

- 「クレーン等安全規則に定められている**事業者**が実施すべき事項」が問われているので、主語が事業者になる文で書くこと。「運転者は〜に注意する」のような書き方は題意とズレます。
- **玉掛け作業（ワイヤロープの点検等）は除外指定**。書くと得点になりません。問題文の「ただし書き」は必ず確認しましょう。
- 2つ求められたら**確実な2つだけ**を簡潔に。たくさん書いても加点はありません。

# 問題9（選択(2)）：重力式擁壁の工程表と全所要日数（記述）

### 問題の概要

現場打ちコンクリート重力式擁壁の構築について、施工手順に基づき工種名を記述した工程表（横線式）を作成し、**全所要日数**を求める問題です。

**条件**：床掘工6日、基礎工3日（床掘工と2日の重複作業）、型枠組立工3日、コンクリート打込み工2日、養生工7日、型枠取外し工1日、埋戻し工2日

### 想定解答

施工手順（工種の順序）：
**床掘工 → 基礎工 → 型枠組立工 → コンクリート打込み工 → 養生工 → 型枠取外し工 → 埋戻し工**

日数の計算：
- 単純合計：6＋3＋3＋2＋7＋1＋2 ＝ 24日
- 基礎工は床掘工と**2日重複**して施工するため：24 − 2 ＝ **22日**

**全所要日数：22日**

バーチャートのイメージ（各工種の開始日）：
- 床掘工：1〜6日目
- 基礎工：5〜7日目（5・6日目は床掘工と並行）
- 型枠組立工：8〜10日目
- コンクリート打込み工：11〜12日目
- 養生工：13〜19日目
- 型枠取外し工：20日目
- 埋戻し工：21〜22日目

### 採点ポイント・よくあるNG

- 計算自体は単純ですが、**「重複2日」を引き忘れて24日と書くNG**が最多パターン。
- 工程表は「工種名を施工手順どおりの順序で記述」できているかが採点対象。**型枠取外し工を養生工の前に置く**などの順序ミスに注意（コンクリートは養生後に脱型が原則）。
- この形式（横線式工程表＋所要日数）は2級で繰り返し出題されている定番。条件の「重複」「待ち日数」の処理だけ練習しておけば確実な得点源になります。
